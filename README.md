# 基于 ThinkPHP 5.0 开发的基本网站管理框架

## 项目链接
**后台登录地址：** http://iqiyi.debuginn.cn/admins.php/admins/account/login

## 项目技术选型
- 操作系统：Linux / Ubuntu16.04
- 编写软件：PHP 7.2
- 数据库：MySQL 5.7.26
- 服务器：Nginx/1.10.3 (Ubuntu)
- 后端框架：ThinkPHP 5.0版本
- 前端框架：Bootstrap 4.1.1
- 组建：Layer，LayUI

## 项目功能实现
1、管理员管理
2、权限管理
3、系统管理
4、标签管理
5、影片管理
6、幻灯片管理
